# ParaCode — Visual Coding for Everyone

## Overview
ParaCode is a browser-based visual coding platform designed to teach programming concepts through interactive block-based coding.  
Users can build logic by dragging and clicking colorful blocks while controlling a robot inside a game world.

The project combines:
- Visual programming
- Interactive gameplay
- Real-time code generation
- Voice commands
- Gamified learning

---

## Features

### 🧩 Block-Based Coding
Users can create programs using visual blocks such as:
- Move Forward
- Move Backward
- Turn Left / Right
- Repeat Loops
- Wait Actions
- Speech Blocks
- Color Trails
- Music Notes
- Celebration Effects

### 🎮 Interactive Game World
- Robot movement on a grid
- Obstacles and walls
- Mission-based levels
- Goal tracking and progress system

### 💻 Live Code View
Automatically converts visual blocks into readable code:
- Python
- JavaScript

### 🎤 Voice Coding
Supports voice commands such as:
- “move forward”
- “turn left”
- “repeat 3”
- “run”

### 🌙 Dark Mode
Includes a complete dark theme for improved user experience.

### 🏅 Gamification
- Levels
- Stars
- Achievements / Badges
- Progress tracking

### 🛠️ Level Creator
Users can:
- Create custom levels
- Add walls and obstacles
- Save levels locally
- Play custom challenges

### 🔊 Text-to-Speech
Optional voice narration for accessibility and guided learning.

## How to Run

1. Download the project files.
2. Open `index.html` in any modern web browser.
3. Start building programs using visual blocks.

No installation or server setup required.
